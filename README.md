# STRIKE — Dice Arena

React와 TypeScript로 만든 싱글플레이 주사위 투기장 게임입니다. 플레이어 1명과 AI 1~3명이 마지막 주사위를 지키기 위해 경쟁합니다.

## 실행 방법

Node.js 18 이상이 필요합니다.

```bash
npm install
npm run dev
```

터미널에 표시되는 로컬 주소(기본값 `http://localhost:5173`)를 브라우저에서 여세요.

## 프로덕션 빌드

```bash
npm run build
npm run preview
```

## 규칙

- 모든 참가자는 주사위 6개로 시작합니다.
- 게임 시작 시 Arena에 X가 아닌 주사위 1개가 놓입니다.
- 자신의 턴에 주사위 1개를 Arena에 던지며, 기존 주사위 하나가 충돌로 재굴림됩니다.
- 충돌 결과를 포함해 같은 눈이 2개 이상 모이면 현재 플레이어가 해당 눈의 주사위를 모두 회수합니다.
- Arena가 비어 있으면 현재 플레이어는 보유 주사위를 모두 던진 뒤 같은 눈 그룹을 회수합니다.
- 같은 눈을 회수하지 못했다면 추가 투척 또는 턴 종료를 선택할 수 있습니다.
- 같은 눈을 회수하면 즉시 턴이 종료됩니다.
- X 처리로 Arena가 비면 현재 턴이 자동 종료되고, 다음 플레이어는 보유 주사위를 모두 던집니다.
- 1은 약 8% 확률로 X 처리되어 게임에서 제외되고 Arena 아래 트레이에 표시됩니다.
- 보유 주사위가 0개가 된 참가자는 탈락하고, 마지막 생존자가 승리합니다.
- AI는 0.8초 후 자동으로 턴을 진행합니다.

AI 난이도별 동작은 `src/game/aiStrategy.ts`에 분리되어 있어 이후 전략을 확장할 수 있습니다.

## 1대1 멀티플레이

호스트 PC에서 서버를 실행합니다.

```bash
npm install
npm run multiplayer
```

터미널에 다음 두 주소가 표시됩니다.

- `Host`: 호스트 PC에서 방을 만들 때 사용
- `Guest`: 같은 네트워크의 다른 사람이 접속할 때 사용

호스트는 `방 만들기`를 선택하고 초대 주소를 전달합니다. 게스트는 해당 주소를 열어 닉네임을 입력합니다. 게스트 닉네임이 대기실에 표시되면 호스트가 게임을 시작할 수 있습니다.

서버가 모든 난수, 턴 권한과 게임 상태를 관리합니다. 멀티플레이 서버가 종료되면 방도 종료됩니다.

### 한 PC에서 테스트

다른 기기 없이 브라우저 탭 두 개로 테스트할 수 있습니다.

1. `npm run multiplayer`를 실행합니다.
2. 첫 번째 탭에서 `http://localhost:8787`을 열고 방을 만듭니다.
3. 두 번째 탭에서 초대 주소의 호스트 부분을 `localhost:8787`로 바꿔 엽니다.
4. 두 번째 탭에서 게스트 닉네임을 입력합니다.
5. 첫 번째 탭에서 닉네임 표시를 확인하고 게임을 시작합니다.

자동 2클라이언트 통합 테스트:

```bash
npm run test:multiplayer
```

인터넷을 통한 접속은 라우터 포트 개방, VPN 또는 별도 서버 배포가 필요합니다.
