# STRIKE — Dice Arena

React와 TypeScript로 만든 싱글플레이 주사위 투기장 게임입니다. 플레이어 1명과 AI 1~3명이 마지막 주사위를 지키기 위해 경쟁합니다.

## 실행 방법

Node.js 18 이상이 필요합니다.

```bash
npm install
npm run dev
```

터미널에 표시되는 로컬 주소(기본값 `http://localhost:5173`)를 브라우저에서 여세요.

## 프로덕션 빌드

```bash
npm run build
npm run preview
```

## 규칙

- 모든 참가자는 주사위 6개로 시작합니다.
- 게임 시작 시 Arena에 X가 아닌 주사위 1개가 놓입니다.
- 자신의 턴에 주사위 1개를 Arena에 던지며, 기존 주사위 하나가 충돌로 재굴림됩니다.
- 충돌 결과를 포함해 같은 눈이 2개 이상 모이면 현재 플레이어가 해당 눈의 주사위를 모두 회수합니다.
- Arena가 비어 있으면 현재 플레이어는 보유 주사위를 모두 던진 뒤 같은 눈 그룹을 회수합니다.
- 같은 눈을 회수하지 못했다면 추가 투척 또는 턴 종료를 선택할 수 있습니다.
- 같은 눈을 회수하면 즉시 턴이 종료됩니다.
- X 처리로 Arena가 비면 현재 턴이 자동 종료되고, 다음 플레이어는 보유 주사위를 모두 던집니다.
- 1은 약 8% 확률로 X 처리되어 게임에서 제외되고 Arena 아래 트레이에 표시됩니다.
- 보유 주사위가 0개가 된 참가자는 탈락하고, 마지막 생존자가 승리합니다.
- AI는 0.8초 후 자동으로 턴을 진행합니다.

AI 난이도별 동작은 `src/game/aiStrategy.ts`에 분리되어 있어 이후 전략을 확장할 수 있습니다.
