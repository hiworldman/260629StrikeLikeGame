# STRIKE — Dice Arena

Vite, React, TypeScript로 만든 브라우저 기반 주사위 투기장 게임입니다. 싱글플레이와 2~5인 온라인 멀티플레이를 지원합니다.

## 실행

Node.js 18 이상이 필요합니다.

```bash
npm install
npm run dev
```

터미널에 표시되는 주소(기본값 `http://localhost:5173`)를 브라우저에서 엽니다.

## 조작

- 자신의 턴에 Arena 안을 원하는 방향으로 스와이프합니다.
- 스와이프 방향이 투척 방향이며, 드래그 거리가 길수록 강하게 던집니다.
- 클릭 또는 `주사위 던지기` 버튼을 사용하면 기본 세기로 정면 투척합니다.
- 모바일에서도 Arena를 터치하고 밀어서 동일하게 조작합니다.

## 물리 및 게임 규칙

- 주사위는 격자가 아닌 Arena의 자유 좌표에 배치됩니다.
- 투척된 주사위는 감속하고, 벽에서 반사되며, 다른 주사위와 충돌합니다.
- 충돌한 두 주사위의 상대속도가 너무 낮으면 눈이 변하지 않습니다.
- 보통 충돌은 일정 확률로 한쪽 눈이, 강한 충돌은 양쪽 눈이 바뀝니다.
- 충돌로 눈이 바뀌는 주사위만 충돌 시점부터 회전 효과가 표시됩니다.
- 모든 주사위가 멈춘 후 X 제거와 같은 눈 회수를 정산합니다.
- X 확률은 15%, 2~6은 각각 17%입니다.
- 시작 주사위는 2명 9개, 3명 8개, 4명 7개, 5명 6개입니다.
- Arena가 비면 다음 플레이어가 보유 주사위를 모두 던집니다.
- 회수한 주사위가 없으면 추가 투척 또는 턴 종료를 선택할 수 있습니다.
- 행동 제한 시간은 7초입니다.

## 프로덕션 빌드

```bash
npm run build
npm run preview
```

## 멀티플레이 서버

```bash
npm run multiplayer
```

호스트가 방을 만든 후 초대 주소를 전달합니다. 게스트는 주소를 열고 닉네임을 입력해 입장합니다. 2~5명이 참가할 수 있으며 시작 플레이어는 게임 시작 전 주사위 굴림으로 결정합니다.

투척 방향, 세기, 물리 결과, 턴 권한은 서버가 판정하고 모든 참가자에게 같은 상태를 전송합니다.

멀티플레이 통합 테스트:

```bash
npm run test:multiplayer
```
